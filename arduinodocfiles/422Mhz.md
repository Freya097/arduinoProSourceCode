# 433MHz RF Wireless Communication

## 📌 Overview

A wireless communication system using 433/422MHz RF transmitter and receiver modules to send data between two Arduino nodes without Wi-Fi or Bluetooth.

---

## 🧰 Components Required

- Arduino Uno (x2)
- 433MHz RF Transmitter Module (FS1000A)
- 433MHz RF Receiver Module (MX-RM-5V)
- Breadboard
- Jumper Wires
- 5V Power Supply

---

## 🔌 Circuit Connections

**Transmitter Side:**
- RF TX Data Pin → Arduino Digital Pin 12
- RF TX VCC → 5V
- RF TX GND → GND

**Receiver Side:**
- RF RX Data Pin → Arduino Digital Pin 11
- RF RX VCC → 5V
- RF RX GND → GND

> ⚠️ Always double-check voltage levels before powering up. Ensure common GND between all modules.

## 📖 Key Definitions

| Term | Definition |
|------|------------|
| **RF (Radio Frequency)** | Electromagnetic waves used for wireless data transmission, operating in the 433MHz ISM band. |
| **Transmitter Module (FS1000A)** | A simple ASK/OOK RF module that converts digital signals into radio waves for wireless broadcast. |
| **Receiver Module (MX-RM-5V)** | Detects RF signals and converts them back to digital data for the microcontroller. |
| **ASK (Amplitude Shift Keying)** | Modulation technique where binary data is encoded as different signal amplitudes. |
| **RadioHead Library** | Arduino library that provides reliable message passing over RF links using the RH_ASK driver. |

## 🚀 How It Works

1. Power up the circuit and upload the Arduino/ESP code.
2. Initialize all sensors and modules in `setup()`.
3. Continuously read sensor data in the `loop()` function.
4. Apply control logic to outputs (relay, motor, display, buzzer).
5. Monitor via Serial Monitor or display for live feedback.
