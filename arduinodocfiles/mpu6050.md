# MPU6050 Accelerometer Gyroscope Interface

## 📌 Overview

Reads 3-axis acceleration and gyroscope data from MPU6050 over I2C and displays processed orientation angles on Serial Monitor or OLED.

---

## 🧰 Components Required

- Arduino Uno
- MPU6050 Module (GY-521)
- OLED Display 0.96" (optional)
- Jumper Wires
- Breadboard

---

## 🔌 Circuit Connections

- MPU6050 SDA → A4 (Arduino SDA)
- MPU6050 SCL → A5 (Arduino SCL)
- MPU6050 VCC → 3.3V
- MPU6050 GND → GND
- MPU6050 INT → Pin 2 (optional interrupt)
- AD0 → GND (I2C address 0x68)

> ⚠️ Always double-check voltage levels before powering up. Ensure common GND between all modules.

---

## 📖 Key Definitions

| Term | Definition |
|------|------------|
| **MPU6050** | 6-axis IMU with MEMS accelerometer (±2/4/8/16g) and MEMS gyroscope (±250/500/1000/2000°/s) on single IC. |
| **DMP (Digital Motion Processor)** | On-chip processor in MPU6050 computing quaternions and Euler angles, offloading calculation from Arduino. |
| **Complementary Filter** | Combining accelerometer and gyroscope data with weighted average to get stable angle without DMP. |
| **Euler Angles** | Roll, Pitch, Yaw angles describing 3D orientation derived from sensor fusion data. |
| **Gyroscope Drift** | Gradual angular error accumulation in gyroscope over time; corrected by accelerometer fusion. |

## 🚀 How It Works

1. Power up the circuit and upload the Arduino/ESP code.
2. Initialize all sensors and modules in `setup()`.
3. Continuously read sensor data in the `loop()` function.
4. Apply control logic to outputs (relay, motor, display, buzzer).
5. Monitor via Serial Monitor or display for live feedback.
